# RunQuest — Version 0.1.0

**Release:** 6 oktober 2025
**Type:** Initial Release
**Status:** Production

---

## Overview

RunQuest är en löparapp för vänner som vill tävla, spåra progress och hålla varandra motiverade. Appen gamifierar löpning via ett XP-system, nivåer, titlar och streaks. Runs kan loggas manuellt eller importeras automatiskt från Strava.

---

## Features

### Authentication
Användare loggar in med användarnamn eller e-post och lösenord. Autentisering hanteras via JWT-tokens som lagras i `localStorage`. Token-livslängd: 7 dagar. Ingen registreringsfunktion för slutanvändare — konton skapas av admin.

### Leaderboard
Visar alla användare rankade efter total XP. Varje användarkort visar:
- Nivå och XP-progress mot nästa nivå
- Total XP och total distans (km)
- Nuvarande streak (dagar i rad med löpning)
- Aktiva titlar
- Profilbild

### XP & Level System
XP tjänas per inloggad löprunda. Beräkningen är konfigurerbar av admin:
- Bas-XP per löpning
- XP per kilometer
- Distansbonusar vid 5 km, 10 km, 15 km, 20 km

Nivåer beräknas på kumulativ XP enligt en progressionskurva. Alla XP-parametrar kan justeras i adminpanelen utan kodredigering.

### Title System
Fyra konkurrenstitlar som delas ut till toppanvändare baserat på statistik. Titlarna uppdateras automatiskt när runs läggs till, redigeras eller tas bort.

| Titel | Kriterie |
|-------|----------|
| Ultra Man | Flest totala kilometer (250+ km) |
| Eliud Kipchoge | Längst enskild löprunda (100+ km) |
| Weekend Destroyer | Högst genomsnittlig distans på helger (12+ km) |
| David Goggins | Längst aktiv streak (20+ dagar) |

### Run Logging
Användare kan logga löpningar manuellt med datum och distans. En löpning genererar XP direkt och uppdaterar användarens totaler, streak och titlar automatiskt. Existerande runs kan redigeras och tas bort.

### Strava Integration
Användare kan koppla sitt Strava-konto via OAuth2. Löpningar importeras automatiskt vid synkronisering. Synk sker antingen manuellt via Settings eller schemalagt. Strava-konto kan kopplas från i Settings.

### User Profiles
Varje användare har en profilsida med:
- Total XP, nivå, total distans, antal löpningar
- Nuvarande och längsta streak
- Alla upplåsta titlar
- Möjlighet att ladda upp profilbild

### Admin Panel
Tillgänglig för användare med adminbehörighet. Funktioner:
- Skapa nya användare
- Återställa lösenord för användare
- Konfigurera XP-parametrar (bas-XP, XP/km, distansbonusar)
- Konfigurera streak-multiplikatorer
- Redigera och ta bort runs för alla användare

### Settings
Användare kan:
- Byta lösenord
- Koppla till/från Strava
- Se Strava-synkstatus

### Mobile Responsive
Gränssnittet är optimerat för mobila enheter med responsiv layout.

## Architecture

| Lager | Teknologi |
|-------|-----------|
| Frontend | React 18, TypeScript, Vite, TailwindCSS, shadcn/ui |
| State | TanStack Query (server state), React Context (auth) |
| Backend | Node.js, Express 5, TypeScript |
| Databas | Supabase (PostgreSQL med Row Level Security) |
| Auth | Eget JWT-system (bcryptjs för lösenord) |
| Strava | OAuth2, REST API |
| Monorepo | npm workspaces |

## Known Limitations

- Ingen självregistrering — konton skapas enbart av admin
- Token refresh ej implementerat — användaren loggas ut efter 7 dagar
- SSL/HTTPS hanteras ej i appkoden — kräver extern reverse proxy
- Inga notifikationer
- Ingen dark mode
